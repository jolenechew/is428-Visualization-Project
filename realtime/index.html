Real-time Earthquake Data Visualization

Visualizing earthquakes in the past week/day using USGS Real Time API




Earthquakes in the Past Week:

This map features earthquakes in the past week, where larger magnitude earthquakes have larger pulse circles and have a darker colour intensity. Earthquakes that cause a tsunami are coloured in blue.


Real Time GPS Based Visualization Last 24hr:

This map features earthquakes in the past day, and was created using Leaflet.js. Explore by clicking on each region to zoom in on the individual earthquakes.

































The data used in this dashboard is obtained by using the API provided by USGS
